# NYAN RENDERER

A basic OpenGL renderer using GLFW, GLAD, GLM, ImGui, nlohmann json etc. libraries.

---

## Building

In project root directory

### Configuring
```
    cmake -S . -B ./build/
```

### Building
```
    cmake --build ./build/
```

### Running
```
    ./build/nyan
```
